## [0.7.0] - 2025-01-30

### 新增

- 📊 Google 模型配额显示 - 在提供商卡片中实时查看各模型剩余配额

- 🎨 全新下拉选择器 - 更美观的自定义下拉组件，告别原生样式

- 🎯 模型选择智能提示 - 配额耗尽的模型显示警告标识

### 修复

- 🔧 修复重启后使用提供商凭证的模型无法使用的问题
- 🔧 修复模型凭证持久化丢失的问题

## 版本号说明

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正
