# 更新日志

所有版本的更新记录。

## [0.6.0] - 2025-01-29

### 新增

- 🔐 提供商管理中心 - 统一管理 15+ AI 服务提供商
- 🔑 OAuth 登录支持 - 支持 GitHub Copilot、OpenAI、Anthropic、Google 账号授权
- 🔄 模型自动获取 - 连接后自动获取提供商最新模型列表
- 🆕 OpenAI 新模型支持 - GPT-5、GPT-4.1、o3 等最新模型

### 改进

- 💾 模型选择记忆 - 切换页面或重启后保留上次选择的模型
- 🔒 Token 自动续期 - OAuth 登录后无需频繁重新授权

## 版本号说明

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正
