# 界面介绍

MobausStudio 采用简洁直观的界面设计，本文将介绍各个功能区域。

## 整体布局

```
┌──────────────────────────────────────────────────────────────┐
│  MobausStudio                                    [─] [□] [×] │
├────────┬─────────────────────────────────────────────────────┤
│        │  ┌─────────────────────────────────────────────┐    │
│  侧    │  │                                             │    │
│  边    │  │              主内容区域                      │    │
│  栏    │  │                                             │    │
│        │  │         显示对话、设置等内容                 │    │
│  导    │  │                                             │    │
│  航    │  │                                             │    │
│        │  └─────────────────────────────────────────────┘    │
│        ├─────────────────────────────────────────────────────┤
│        │  [输入区域]                              [发送]     │
└────────┴─────────────────────────────────────────────────────┘
```

---

## 侧边栏

侧边栏位于界面左侧，提供主要功能的快速访问：

### 主导航区

| 图标 | 功能 | 说明 |
|------|------|------|
| 💬 | 对话 | AI 对话界面 |
| 🤖 | 智能体 | Agent 管理，创建和配置 AI 助手 |
| 🧩 | 技能 | 预设提示词模板 |
| 🔌 | MCP | MCP 服务管理 |
| 🖥️ | 模型 | AI 模型配置和管理 |

### 底部功能区

| 图标 | 功能 | 说明 |
|------|------|------|
| 📊 | 统计 | 使用统计信息 |
| ⚙️ | 设置 | 应用配置 |

点击图标即可切换到对应功能页面。

---

## 对话界面

### 对话列表

左侧显示历史对话列表：
- 点击对话可切换
- 点击 `+` 新建对话
- 右键可删除或重命名

### 消息区域

中间区域显示对话内容：
- **用户消息**: 显示在右侧
- **AI 回复**: 显示在左侧，支持 Markdown 渲染
- **代码块**: 语法高亮，可一键复制

### 输入区域

底部输入框：
- 输入消息后按 `Enter` 发送
- `Shift + Enter` 换行
- 支持粘贴图片（部分模型支持）

### 模型选择

顶部可以选择当前使用的 AI 模型。

## 各功能页面说明

### 模型页面（独立页面）

点击侧边栏「🖥️ 模型」进入，用于配置 AI 模型：

- 添加/编辑/删除模型配置
- 设置 API 密钥和端点
- 测试模型连接状态
- 支持 OpenAI、Anthropic、自定义兼容端点

### 智能体页面（独立页面）

点击侧边栏「🤖 智能体」进入，用于管理 AI 助手：

- 创建自定义 Agent
- 配置系统提示词
- 绑定技能和 MCP 工具
- 运行 Agent 开始对话

### MCP 页面（独立页面）

点击侧边栏「🔌 MCP」进入，管理 MCP 服务器连接：

- 添加/删除服务器
- 查看连接状态和工具列表
- 配置 stdio/HTTP 传输方式
- 设置自动启动

### 设置页面

点击侧边栏「⚙️ 设置」进入：

- 通用设置：主题切换（浅色/深色）、语言设置
- 数据管理：导入/导出配置
- 关于：版本信息、检查更新、开源许可

---

## 快捷操作

### 消息操作

鼠标悬停在消息上会显示操作按钮：
- 📋 复制内容
- 🔄 重新生成（AI 回复）
- ✏️ 编辑（用户消息）

### 代码块操作

代码块右上角提供：
- 📋 复制代码
- 显示语言类型

---

## 主题

MobausStudio 支持浅色和深色主题：

- **浅色主题**: 适合日间使用
- **深色主题**: 适合夜间使用，减少眼睛疲劳

在「设置」→「通用」中切换主题。
