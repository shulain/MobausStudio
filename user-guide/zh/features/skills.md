# 技能系统

技能是预定义的提示词模板，帮助你快速完成特定任务，无需每次都编写复杂的提示词。

## 什么是技能？

技能 = 预设提示词 + 参数模板

例如，「代码审查」技能可能包含：
- 预设的审查标准
- 输出格式要求
- 关注点提示

你只需要提供代码，技能会自动组装完整的提示词。

---

## 使用技能

### 1. 打开技能页面

点击侧边栏的「🧩 技能」图标。

### 2. 选择技能

浏览可用技能列表，点击选择要使用的技能。

### 3. 填写参数

根据技能要求填写必要的参数（如代码、文本等）。

### 4. 执行

点击「执行」或「发送」，技能会自动生成完整提示词并发送给 AI。

## 内置技能

MobausStudio 提供了 3 个内置技能，开箱即用：

| 技能 | 功能 | 说明 |
|------|------|------|
| 代码审查 | 审查代码质量、安全性、性能 | 按 P0/P1/P2 优先级分类问题，提供专业审查报告 |
| 翻译专家 | 多语言翻译 | 支持中/英/日/韩，遵循信达雅原则 |
| 写作助手 | 文章润色、扩写、改写 | 支持商务邮件、技术文档、营销文案等多种场景 |

> 💡 内置技能不可编辑或删除，但你可以创建自定义技能来满足更多需求。

---

## 技能参数

不同技能可能需要不同的参数：

### 必填参数

技能运行必需的输入，如：
- 代码审查：需要提供代码
- 翻译：需要提供原文和目标语言

### 可选参数

可以自定义的选项，如：
- 输出语言
- 详细程度
- 特定关注点

---

## 自定义技能

### 创建技能

1. 点击侧边栏的「🧩 技能」图标进入技能页面
2. 点击「新建技能」按钮
3. 填写技能信息：
   - 名称
   - 描述
   - 提示词模板
   - 参数定义

### 提示词模板语法

使用 `{{参数名}}` 作为占位符：

```
请审查以下 {{language}} 代码：

{{code}}

重点关注：
- 代码质量
- 安全性
- {{focus}}
```

## 安装与导出技能

### 安装技能

你可以从官方仓库或本地文件安装技能：

1. 点击「安装技能」按钮
2. 选择安装来源：
   - **官方仓库**：浏览并安装官方提供的技能
   - **本地文件**：导入 JSON 格式的技能文件
3. 选择要安装的技能，点击「安装」

> 💡 **提示**：如果技能已存在，可以选择更新现有技能。

### 导出技能

将自定义技能导出为 JSON 文件，方便分享和备份：

1. 点击「导出」按钮
2. 选择要导出的技能
3. 点击「导出」保存文件

### 删除技能

自定义技能可以删除：

1. 在技能卡片上点击「删除」按钮
2. 确认删除操作

> ⚠️ 内置技能不可删除。

## 使用技巧

### 1. 组合使用

可以先用一个技能处理，再用另一个技能优化结果。

### 2. 调整参数

根据实际需求调整可选参数，获得更精准的结果。

### 3. 迭代优化

如果结果不满意，可以修改参数重新执行，或在对话中继续优化。
