# 快捷键

掌握快捷键可以提高使用效率。

## 对话快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift + Enter` | 在输入框中换行（不发送） |

---

## 系统快捷键

以下是操作系统和浏览器提供的通用快捷键，在 MobausStudio 中同样适用：

### 编辑操作

| 快捷键 | macOS | Windows/Linux | 功能 |
|--------|-------|---------------|------|
| 复制 | `⌘ + C` | `Ctrl + C` | 复制选中内容 |
| 粘贴 | `⌘ + V` | `Ctrl + V` | 粘贴内容 |
| 剪切 | `⌘ + X` | `Ctrl + X` | 剪切选中内容 |
| 全选 | `⌘ + A` | `Ctrl + A` | 全选内容 |
| 撤销 | `⌘ + Z` | `Ctrl + Z` | 撤销操作 |
| 重做 | `⌘ + ⇧ + Z` | `Ctrl + Shift + Z` | 重做操作 |

### 窗口操作（桌面版）

| 快捷键 | macOS | Windows/Linux | 功能 |
|--------|-------|---------------|------|
| 关闭窗口 | `⌘ + W` | `Ctrl + W` | 关闭当前窗口 |
| 退出应用 | `⌘ + Q` | `Alt + F4` | 退出应用程序 |
| 最小化 | `⌘ + M` | `Win + ↓` | 最小化窗口 |
| 全屏 | `⌘ + Ctrl + F` | `F11` | 切换全屏模式 |
| 放大 | `⌘ + +` | `Ctrl + +` | 放大界面 |
| 缩小 | `⌘ + -` | `Ctrl + -` | 缩小界面 |
| 重置缩放 | `⌘ + 0` | `Ctrl + 0` | 重置为默认大小 |

---

## 输入技巧

### 中文输入法

使用中文输入法时，按 `Enter` 确认选词不会发送消息，只有在非输入法状态下按 `Enter` 才会发送。

### 粘贴图片

直接在输入框中粘贴图片（`Ctrl/Cmd + V`），图片会自动添加为附件。

### 拖拽文件

将文件拖拽到对话窗口，可以快速添加附件。
