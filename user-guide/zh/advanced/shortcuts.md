# 快捷键

掌握快捷键可以大幅提高使用效率。

## 通用快捷键

| 快捷键 | macOS | Windows/Linux | 功能 |
|--------|-------|---------------|------|
| 新建对话 | `⌘ + N` | `Ctrl + N` | 创建新的对话 |
| 打开设置 | `⌘ + ,` | `Ctrl + ,` | 打开设置页面 |
| 关闭窗口 | `⌘ + W` | `Ctrl + W` | 关闭当前窗口 |
| 退出应用 | `⌘ + Q` | `Alt + F4` | 退出应用程序 |

---

## 对话快捷键

| 快捷键 | macOS | Windows/Linux | 功能 |
|--------|-------|---------------|------|
| 发送消息 | `Enter` | `Enter` | 发送当前输入 |
| 换行 | `⇧ + Enter` | `Shift + Enter` | 在输入框中换行 |
| 聚焦输入框 | `⌘ + L` | `Ctrl + L` | 快速定位到输入框 |
| 清空输入 | `⌘ + ⇧ + Backspace` | `Ctrl + Shift + Backspace` | 清空输入框内容 |

---

## 编辑快捷键

| 快捷键 | macOS | Windows/Linux | 功能 |
|--------|-------|---------------|------|
| 复制 | `⌘ + C` | `Ctrl + C` | 复制选中内容 |
| 粘贴 | `⌘ + V` | `Ctrl + V` | 粘贴内容 |
| 剪切 | `⌘ + X` | `Ctrl + X` | 剪切选中内容 |
| 全选 | `⌘ + A` | `Ctrl + A` | 全选内容 |
| 撤销 | `⌘ + Z` | `Ctrl + Z` | 撤销操作 |
| 重做 | `⌘ + ⇧ + Z` | `Ctrl + Shift + Z` | 重做操作 |

---

## 导航快捷键

| 快捷键 | macOS | Windows/Linux | 功能 |
|--------|-------|---------------|------|
| 上一个对话 | `⌘ + ↑` | `Ctrl + ↑` | 切换到上一个对话 |
| 下一个对话 | `⌘ + ↓` | `Ctrl + ↓` | 切换到下一个对话 |
| 滚动到顶部 | `⌘ + Home` | `Ctrl + Home` | 滚动到对话顶部 |
| 滚动到底部 | `⌘ + End` | `Ctrl + End` | 滚动到对话底部 |

---

## 窗口快捷键

| 快捷键 | macOS | Windows/Linux | 功能 |
|--------|-------|---------------|------|
| 最小化 | `⌘ + M` | `Win + ↓` | 最小化窗口 |
| 全屏 | `⌘ + Ctrl + F` | `F11` | 切换全屏模式 |
| 放大 | `⌘ + +` | `Ctrl + +` | 放大界面 |
| 缩小 | `⌘ + -` | `Ctrl + -` | 缩小界面 |
| 重置缩放 | `⌘ + 0` | `Ctrl + 0` | 重置为默认大小 |

---

## 自定义快捷键

目前暂不支持自定义快捷键，该功能正在开发中。

---

## 快捷键冲突

如果快捷键与系统或其他应用冲突：

1. 检查系统快捷键设置
2. 关闭冲突的应用
3. 使用替代操作（鼠标点击）
