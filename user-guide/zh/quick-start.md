# 快速入门

本指南将帮助你在 5 分钟内开始使用 MobausStudio。

---

## 第一步：配置 AI 模型

首次启动 MobausStudio 后，你需要配置至少一个 AI 模型才能开始对话。

1. 点击左侧边栏的 **⚙️ 设置** 图标
2. 选择 **模型配置**
3. 添加你的 API 密钥：
   - **OpenAI**: 输入你的 OpenAI API Key
   - **Anthropic**: 输入你的 Claude API Key
   - **其他兼容模型**: 配置自定义 API 端点

> 💡 **提示**: 如果你还没有 API 密钥，可以访问对应服务商官网申请。

---

## 第二步：开始对话

1. 点击左侧边栏的 **💬 对话** 图标
2. 在底部输入框输入你的问题
3. 按 `Enter` 或点击发送按钮
4. 等待 AI 回复

### 对话技巧

- **多轮对话**: AI 会记住上下文，你可以进行连续对话
- **新建对话**: 点击 `+` 按钮开始新的对话
- **切换模型**: 在对话界面顶部可以切换不同的 AI 模型

---

## 第三步：探索更多功能

### MCP 服务（可选）

MCP (Model Context Protocol) 允许 AI 连接外部工具和服务：

1. 进入 **设置** → **MCP 配置**
2. 添加 MCP 服务器配置
3. 在对话中，AI 将能够使用这些工具

### 技能系统（可选）

技能是预定义的提示词模板，帮助你快速完成特定任务：

1. 点击 **🎯 技能** 图标
2. 浏览可用技能
3. 点击使用，自动填充提示词

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行（不发送） |
| `Ctrl/Cmd + N` | 新建对话 |
| `Ctrl/Cmd + ,` | 打开设置 |

更多快捷键请查看 [快捷键列表](./advanced/shortcuts.md)
