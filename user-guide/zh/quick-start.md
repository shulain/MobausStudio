# 快速入门

本指南将帮助你在 5 分钟内开始使用 MobausStudio。

---

## 第一步：连接 AI 提供商

首次启动 MobausStudio 后，你需要先连接一个 AI 服务提供商。

### 方式一：OAuth 登录（推荐）

如果你有 ChatGPT Plus/Pro、Claude Pro/Max 或 GitHub Copilot 订阅：

1. 点击左侧边栏的 **🔗 提供商** 图标
2. 找到对应的提供商（如 OpenAI、Anthropic、GitHub Copilot）
3. 点击「连接」，选择「OAuth 登录」
4. 在浏览器中完成授权
5. 授权成功后自动返回应用

### 方式二：API Key

如果你有 API Key：

1. 点击左侧边栏的 **🔗 提供商** 图标
2. 找到对应的提供商，点击「连接」
3. 选择「API Key」方式
4. 输入你的 API Key，点击「连接」

> 💡 **提示**: 如果你还没有 API 密钥，可以点击「获取 API Key」链接访问对应服务商官网申请。

---

## 第二步：添加模型

连接提供商后，添加你想使用的模型：

1. 点击左侧边栏的 **🖥️ 模型** 图标
2. 点击「添加模型」
3. 选择已连接的提供商
4. 选择具体模型（如 GPT-4o、Claude 3.5 Sonnet）
5. 点击「保存」

---

## 第三步：开始对话

1. 点击左侧边栏的 **💬 对话** 图标
2. 在底部输入框输入你的问题
3. 按 `Enter` 或点击发送按钮
4. 等待 AI 回复

### 对话技巧

- **多轮对话**: AI 会记住上下文，你可以进行连续对话
- **新建对话**: 点击 `+` 按钮开始新的对话
- **切换模型**: 在对话界面顶部可以切换不同的 AI 模型

---

## 第四步：探索更多功能

### 智能体（可选）

创建专属的 AI 助手：

1. 点击 **🤖 智能体** 图标
2. 创建新智能体，配置系统提示词
3. 绑定技能和 MCP 工具

### MCP 服务（可选）

MCP (Model Context Protocol) 允许 AI 连接外部工具和服务：

1. 点击 **🔌 MCP** 图标
2. 添加 MCP 服务器配置
3. 在对话中，AI 将能够使用这些工具

### 技能系统（可选）

技能是预定义的提示词模板，帮助你快速完成特定任务：

1. 点击 **🧩 技能** 图标
2. 浏览可用技能
3. 点击使用，自动填充提示词
